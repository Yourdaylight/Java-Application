Main为主函数，通过调用下述5个工具类以实现百宝箱的功能。运行Main以启动整个项目
#######
生活百宝箱共五个工具类对应五个功能，其中
IpUtil:实现ip地址查询功能，通过调用太平洋网的查询接口，解析返回的json数据得到地址信息

WeatherUtil:实现天气预报查询，通过调用webService框架得到

jianfanConvert:实现简繁体转换，已经将所有简体与繁体的对应字符串保存在代码中，用户根据需求选择简体转繁体或者繁体转简体。

IdCardUtil:实现身份证归属地查询。将所有的省市区县信息的编码保存在HashMap中，以此来对身份证进行解析。
